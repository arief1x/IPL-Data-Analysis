# IPL-Data-Analysis (2008-2024)
## Overview
Exploratory Data Analysis of IPL matches using Python and data visualization libraries.

## Dataset
- 1095 matches | 2,60,920 ball-by-ball records
- Source : Kaggle

## Key Insights
- 🏆 Mumbai Indians - most wins in IPL history
- 🎲 Toss winning has minimal impact (50.6% win rate)
- 🌟 Ab de Villiers - most Player of Match awards
- 📈 IPL scoring grew 45% from 2008 to 2024
- 🏏 Batting first has slight edge (50.9%)

## Tools Used
Python | Pandas | Numpy | Seaborn | Matplotlib
